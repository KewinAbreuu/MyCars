# Bem-vindo ao Mycars! </hr>
<p>O Mycars é um projeto baseado em um CRUD de veículos.<p>

## 💻 Funcionalidades</hr>
<p>-Cadastar</p>
<p>-Listar</p>
<p>-Apagar</p>
<p>-Editar</p>
<p>-Buscar</p>

## 🚀  Como usar! </hr>
Após clonar o repository Mycars, na pasta Mycars (font-end)
```
  $ npm i
```

Start no front-end :
```
  $ npm start
```
